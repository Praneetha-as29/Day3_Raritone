# AI/ML Internship — Day 3

## Overview
This project demonstrates human pose detection and basic body measurement logic using computer vision. It also includes research on AI applications in fashion such as virtual try-on systems and recommendation models.

---

## Tasks Completed

### Task 1 — Human Pose Detection
- Implemented using OpenCV and MediaPipe
- Detected body landmarks
- Displayed skeleton tracking in real time

---

### Task 2 — Body Measurement Logic
- Studied:
  - Shoulder width calculation
  - Height estimation
  - Waist estimation
- Used landmark-based distance calculation

---

### Task 3 — AI Research
- Studied:
  - Virtual try-on systems
  - Fashion recommendation models
